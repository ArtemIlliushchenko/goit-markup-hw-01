# goit-markup-hw-01

📌 WebStudio — One Page Project

Односторінковий адаптивний сайт, зібраний на основі виконаних домашніх завдань.
Проєкт реалізує повноцінну landing page структуру з інтерактивними елементами та адаптивною версткою.

🔴 Live Demo:[(посилання)](https://artemilliushchenko.github.io/goit-markup-hw-01/)

📖 Опис

Hero-секцію

Секцію переваг

Секцію команди

Портфоліо

Footer

Мобільне меню

Модальне вікно з формою

Проєкт побудований з використанням семантичної HTML-розмітки та modern CSS-підходів.
Сайт повністю адаптований для мобільних пристроїв, планшетів та десктопів.
